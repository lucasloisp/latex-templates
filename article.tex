\section{Programming}

Inline code samples: \mintinline{c}{int} or function calls like \mintinline{python}{print(x**2)}.
We also allow code blocks, as follows, or even floating listings (\ref{l:python-hello-world}).

\begin{minted}{c}
int main() {
    printf("hello, world");
    return 0;
}
\end{minted}

\begin{listing}
\begin{minted}{python}
def main():
    print("hello, world")
    return 0
\end{minted}
